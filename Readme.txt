MonitorLogFolders
=================
PowerShell Script that provides a menu system for exporting mailboxes from Exchange 2007.


Please see: http://it-erate.com/exporting-exchange-2007-mailboxes-powershell/ for further details on how to configure this script.


Main Files
=================
ExportToPst.ps1 - Only file needed.


Usage
==================
Just run the script and all the heavy lifting is done by the script.
